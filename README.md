# Ecommerce Product Management

## Project Overview
This project is designed to manage e-commerce products efficiently. It allows users to add, update, delete, and view products in an organized manner. The application is built using C++ and utilizes various data structures to enhance performance and usability.

## Setup Instructions
1. **Clone the repository**:
   ```bash
   git clone https://github.com/aryananmol18/ecommerce-product-management-cpp.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd ecommerce-product-management-cpp
   ```

3. **Compile the source code**:
   Ensure you have a C++ compiler installed (like g++):
   ```bash
   g++ main.cpp -o ecommerce_product_management
   ```

4. **Run the application**:
   ```bash
   ./ecommerce_product_management
   ```

5. **Follow the on-screen instructions** to manage your products.
